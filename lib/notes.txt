 🧠 What is ?? ?

?? means:

👉 “If left side is NOT null → use it”
👉 “If left side is null → use right side”

🍕 Real-life example
String? name;

String displayName = name ?? "Guest";
What happens:
If name = "Tanmay" → result = "Tanmay" ✅
If name = null → result = "Guest" 👤
